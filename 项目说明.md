# 🚗 雨刷尺寸查询系统 - 项目说明

## 📂 项目文件结构

```
Contacts/
├── app.py                    # ⭐ Streamlit 主应用程序
├── wiper_data.xlsx          # 📊 雨刷数据 Excel 文件
├── requirements.txt         # 📦 Python 依赖包列表
├── README.md               # 📖 项目完整说明文档
├── DEPLOY.md              # 🚀 详细部署指南
├── 快速部署指南.md        # ⚡ 快速部署步骤
├── 项目说明.md            # 📝 本文件
├── create_sample_data.py  # 🔧 示例数据生成脚本
├── start.bat              # 🪟 Windows 启动脚本
├── start.sh               # 🐧 Linux/Mac 启动脚本
├── .gitignore             # 🚫 Git 忽略文件配置
└── .streamlit/
    └── config.toml        # ⚙️ Streamlit 配置文件
```

## 🎯 核心功能

1. **车型搜索**：输入车型名称，模糊匹配查询
2. **数据展示**：显示品牌、车型、年份、主驾、副驾、接头、后雨刷
3. **统计信息**：显示品牌数量、车型数量
4. **响应式设计**：自动适配手机、平板、电脑

## 📊 数据格式

Excel 文件（`wiper_data.xlsx`）包含以下列：

| 列名 | 说明 | 示例 |
|------|------|------|
| 品牌 | 汽车品牌 | 奥迪、宝马、奔驰 |
| 车型 | 车型名称 | A4、X5、凯美瑞 |
| 年份 | 适用年份 | 2020-2023 |
| 主驾 | 主驾驶侧雨刷尺寸 | 24英寸 |
| 副驾 | 副驾驶侧雨刷尺寸 | 19英寸 |
| 接头 | 雨刷接头类型 | U型、J型 |
| 后雨刷 | 后雨刷尺寸 | 14英寸 或 - |

## 🚀 三种使用方式

### 方式 1：本地运行（开发测试）

**Windows:**
```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

或直接双击 `start.bat`

**Mac/Linux:**
```bash
pip install -r requirements.txt
streamlit run app.py
```

或运行 `./start.sh`

访问：http://localhost:8501

### 方式 2：GitHub + Streamlit Cloud 部署（推荐）

这是最简单的部署方式，完全免费！

1. **推送到 GitHub**
```bash
git init
git add .
git commit -m "雨刷尺寸查询系统"
git branch -M main
git remote add origin https://github.com/你的用户名/仓库名.git
git push -u origin main
```

2. **在 Streamlit Cloud 部署**
   - 访问：https://share.streamlit.io
   - 用 GitHub 登录
   - 点击 "New app"
   - 选择仓库和 `app.py`
   - 点击 "Deploy"

3. **完成！**
   - 等待 2-5 分钟
   - 获得公开访问的 URL

## 🔄 更新数据

### 方法 1：直接编辑 Excel 文件

1. 打开 `wiper_data.xlsx`
2. 编辑数据（保持列名不变）
3. 保存文件
4. 如果是云端部署：
```bash
git add wiper_data.xlsx
git commit -m "更新数据"
git push
```

### 方法 2：使用脚本重新生成

编辑 `create_sample_data.py`，然后运行：
```bash
python create_sample_data.py
```

## 🔧 自定义配置

### 修改界面样式

编辑 `app.py` 中的 CSS 样式部分：

```python
st.markdown("""
<style>
    /* 在这里修改样式 */
    h1 {
        color: #1f77b4;  /* 标题颜色 */
    }
</style>
""", unsafe_allow_html=True)
```

### 修改主题

编辑 `.streamlit/config.toml`：

```toml
[theme]
primaryColor="#1f77b4"      # 主色调
backgroundColor="#FFFFFF"    # 背景色
secondaryBackgroundColor="#f0f2f6"  # 次背景色
textColor="#262730"         # 文字颜色
```

## 🐛 故障排除

### 问题：找不到数据文件
```bash
# 检查文件是否存在
ls wiper_data.xlsx

# 确保文件在项目根目录
```

### 问题：中文显示乱码
- 确保所有文件使用 UTF-8 编码保存
- Excel 文件另存为时选择 UTF-8 编码

### 问题：部署失败
- 检查 requirements.txt 是否正确
- 确保所有依赖版本兼容
- 查看 Streamlit Cloud 部署日志
